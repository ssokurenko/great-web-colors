## Local project setup

### Install Bower and the project dependencies

`npm install -g bower`

### Install Grunt:

`npm install -g grunt`

### Grunt commands

Run local server:

`grunt server`

The project will be available at: http://127.0.0.1:9000/

Run build script for distribution:

`grunt build`
